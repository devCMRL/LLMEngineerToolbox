Starter stack: local RAG

Good for solo builders, research labs, and quick internal tools.

  1. Docling or PyMuPDF4LLM for PDF extraction
  2. Chroma or Qdrant for retrieval
  3. LlamaIndex or Haystack for RAG logic
  4. Ollama or vLLM for local serving
  5. Langfuse or Phoenix for tracing

Starter stack: production agent app

Good for multi-step assistants, enterprise copilots, and tool-using workflows.

  1. Unstructured or Docling for ingestion
  2. Pinecone, Weaviate, or Qdrant for managed retrieval
  3. LangGraph, CrewAI, or OpenAI Agents SDK for agent logic
  4. Temporal or Prefect for durable orchestration
  5. LiteLLM + vLLM/TGI for model routing and serving
  6. LangSmith, Phoenix, or MLflow for evals and observability
  7. NeMo Guardrails + MCP where governance and tools matter

Decision heuristic

Use this shortcut when choosing among the major categories.

  1. Need batteries-included RAG? Start with LlamaIndex or Haystack.
  2. Need fine-grained, stateful agent control? Start with LangGraph.
  3. Need durable long-running jobs? Add Temporal or Prefect.
  4. Need local inference? Start with Ollama or vLLM.
  5. Need evals before launch? Add Promptfoo, Langfuse, Phoenix, or LangSmith.
